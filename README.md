# 📝 轻量级高颜值问卷系统 (EasySurvey)

[![PHP Version](https://img.shields.io/badge/php-%3E%3D%208.0-blue.svg)](https://php.net)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/)

这是一款基于 **PHP 8.x + MySQL** 开发的极简、轻量级问卷调查系统。项目零外部复杂依赖（无需 Composer 依赖与 Node.js 运行时），极其适合**单站点部署、企业内部满意度调研、小规模数据收集**等应用场景。

本项目在品牌定制、题号配置与二维码生成上的核心功能设计，采用现代高颜值的毛玻璃玻璃拟态（Glassmorphism）与多维度动态渐变样式，提供开箱即用的优质体验。

---

## 🌟 核心特性

- 📊 **多功能仪表盘**：后台一键掌控问卷总数、启用问卷数、回答总数及今日收集动态。
- 🎨 **极致品牌定制**：支持上传系统/问卷专属 Logo，自由定制主题色、背景色、问卷头图、背景图以及提交按钮文本。
- 📝 **可视化问卷编辑器**：支持单选题、多选题、文本题三种经典题型，可自由拖拽/上下移稳定排序，题号智能关联，**并提供一键导出问卷 CSV 结构及导入本地 Excel/CSV 格式问卷**，极大方便大批量题目的快速编辑。
- 🛡️ **安全物理保留防误删**：重构了传统问卷系统编辑导致历史答案丢失的缺陷。编辑已收集回答的问卷时，仅对明确删除的问题进行安全同步，并在前端加入防误删强警示。
- 📈 **数据可视化与导出**：内置本地离线版 **Chart.js** 柱状分析图，完美生成占比统计表格，并提供一键导出为通用 UTF-8 CSV 报表功能。
- 📱 **二维码与预览**：自动根据配置的站点基础链接生成专属扫码卡片，支持一键在浏览器端导出高清晰度二维码 PNG 贴图。
- 🔒 **多重安全防护**：
  - 全面支持 Cookie 级别防重复提交及基于客户端 IP 的智能频控。
  - 核心逻辑内置细致的 CSRF 安全校验，物理隔离数据库敏感凭据。
  - **严防 XSS 攻击**：物理禁用 SVG 等 XML 文件上传，加强后台接口对非法 Question ID 投递以及选项篡改的深度鉴权拦截。

## 🛠️ 运行环境要求

- **PHP** >= 8.0
- **MySQL** >= 8.0 / 5.7
- PHP 核心扩展：`PDO`, `pdo_mysql`, `gd`, `session`, `mbstring`
- Web 服务器（如 Apache / Nginx / IIS / PHP 内置 Server）
- 物理可写目录：`public/uploads/` (子目录在运行时会自动创建)

## 🚀 快速开始

### 第一步：获取代码并忽略配置
克隆项目到本地 Web 根目录下：
```bash
git clone https://github.com/aqppy/EasySurvey.git
```

### 第二步：初始化配置文件
系统采用私有配置隔离，请在项目根目录下，将模板复制为本地私有配置文件：
```bash
cp app/config.example.php app/config.php
```
使用您喜爱的编辑器打开 `app/config.php`，并根据说明配置您的 **MySQL 数据库参数** 与 **管理员强密码哈希**（文件内有生成示例）。

### 第三步：导入数据库结构
使用 MySQL 客户端，在对应的数据库中运行项目根目录下的 [database.sql](database.sql) 初始化 7 张核心表。

### 第四步：部署并开始填写
- **Web 根目录指向**：为了保护敏感逻辑，请确保将您 Web 服务器的根目录指向项目的 **`public/`** 子目录，绝不可暴露项目根目录！
- **本地测试运行**：您可以直接在项目根目录下通过 PHP 内置开发服务器快速启动测试：
  ```bash
  php -S 127.0.0.1:8089 -t public
  ```
- 打开浏览器访问 `http://127.0.0.1:8089/check_env.php` 即可进行一键环境诊断。
- 诊断全部通过后，即可访问 `http://127.0.0.1:8089/admin/login.php` 登录进入后台（初始账号：`admin`，初始密码参见您在 `config.php` 中配置的管理员账号说明）。

## ⚙️ 实用管理工具

本项目为开源运维人员编写了若干命令行及图形化诊断辅助工具：

### 1. 数据库一键本地备份
在终端执行以下命令，即可直接在 `backups/` 目录下生成包含 `DROP TABLE`、`CREATE TABLE` 以及 `INSERT INTO` 的完整结构和数据 SQL 备份快照（基于 pure PHP PDO，免除了对物理 `mysqldump` 依赖的限制）：
```bash
php -f scripts/backup.php
```

### 2. 自动化一致性回归测试
为了保证您二次开发后系统的逻辑稳定性，您可以直接运行我们提供的自动化闭环测试包：
```bash
php -f tests/regression.php
```
*注：该测试会在数据库事务内完成 mock 数据的完整周期流转与断言检测，并在通过后无痛自动回滚事务，不会在您的生产环境中留下任何多余的脏数据。*

### 3. 一键环境自检页面
部署完毕或系统发生异常时，可以直接在浏览器中访问系统自带的诊断大屏 `http://您的域名/check_env.php`。它会以极致美观的高透玻璃拟态大屏完整诊断 PHP 版本、核心组件启用状态、素材上传文件夹写权限以及数据库联通细节。
