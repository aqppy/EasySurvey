## 1. 🎨 仪表盘 (Dashboard)
登录后台后，首屏即为**仪表盘**。它为您提供系统全貌的实时动态数据：
* **核心指标卡片**：
  * **问卷总数**：系统中已创建的问卷总量。
  * **启用中问卷**：当前正在前台接受填写的问卷数。
  * **收集回答数**：系统自上线以来累计收集到的答卷总份数。
* **快捷入口**：点击卡片可快速跳转至相应的管理页面。

## 2. 📝 问卷管理与智能编辑器
点击导航栏的 **「问卷管理」** 进入问卷列表。在这里您可以管理所有问卷的生命周期。

### 设计新问卷
点击 **「创建新问卷」** 即可打开可视化问卷编辑器：
1. **基础信息**：填写**问卷标题**和**问卷说明**（支持留空）。
2. **题目设计**：点击底部的 **「+ 添加题目」** 按钮：
   * **题目类型**：支持 **单选题**、**多选题** 和 **文本题**（问答题）。
   * **选项管理**：对于单选和多选，可自由添加或移除选项，支持拖拽和移动。
   * **属性控制**：可勾选是否为 **「必填」** 项。
3. **题目排序**：点击题目右上角的 **`↑`** 或 **`↓`** 按钮，可在前端及数据库中实时调整该题目的展现顺序。
4. **个性化主题设置**（在编辑器右侧/下方）：
   * **问卷 Logo**：可上传企业品牌徽标（支持 JPG、PNG、WEBP，大小限制 2MB）。
   * **问卷头图**：上传置于问卷顶部的视觉大图。
   * **问卷 background 图**：可上传平铺背景。
   * **提交按钮文本**：个性化定制答题按钮文字（例如：`立即提交`、`确认并发送`）。
   * **显示控制**：支持一键开关是否在前台展示 `问卷标题`、`问卷说明` 和 `题目序号`。

### 安全物理删除保障
* **题目级联保护**：当您编辑一份已经收集到答卷的历史问卷时，如果尝试在前端删除某个旧题目，系统会触发**强制红色防误删确认警告**。防止管理员意外物理删除题目而连带清空已收集的历史答案。

### ⚡ 高阶特性：CSV 结构导入与导出
为了方便您在本地批量编辑问卷，或将问卷结构在不同部署环境之间流转，本系统提供了强大的 **CSV 导入导出** 功能：

#### 1. 导出问卷结构 (CSV)
在问卷列表的每条问卷右侧，点击 **「导出 CSV 结构」**：
* 系统会自动生成包含该问卷所有题目、题型、必填状态、选项的 CSV 文件。
* **Excel 友好支持**：导出的 CSV 文件**内置 UTF-8 BOM 签名**。在 Windows 环境下双击使用 Microsoft Excel 打开**绝对不会产生中文乱码**。

#### 2. 本地一键导入问卷 (CSV)
在编辑问卷页面，点击 **「导入 CSV 问卷」** 按钮并选择本地的 CSV 文件：
* **高度编码兼容**：导入器拥有智能编码识别，完美兼容 `UTF-8` 与 `GBK / GB2312`。
* **高宽容选项分割**：选项一栏支持常见的 `|` 字符进行严格拆分。
* **安全预览机制**：CSV 解析出的题目会**即时呈现在网页编辑器中**供您预览和精细微调（此时尚未保存到数据库），确认无误后拉到页面最下方点击 **「保存问卷」** 方可正式入库，操作链路极具安全感。

## 3. 📊 数据查看与统计分析
点击导航栏的 **「数据查看」**。这是收集阶段结束后，管理员最核心的分析控制台。

### 数据报表与可视化
* **Chart.js 极致离线驱动**：系统内置了完全本地化的图形渲染库。即使部署在完全与外网隔离的纯内网或 VPS 专网环境下，依然能渲染出动感、彩色的饼图和柱状图，实时统计每道单选题 and 多选题的选项占比。
* **答卷明细列表**：下方以表格形式按时间倒序呈现每一份答卷的提交 IP 地址、提交时间。点击 **「查看详情」** 可查阅该答卷的所有题目答案明细。

### 明细数据导出
* 点击右上角 **「导出数据 (CSV)」**，可将当前问卷收集到的所有回答明细（一行一条记录，包含提交时间、IP、每道题的回答内容）一键导出为 CSV 电子表格，方便导入 SPSS、Excel、R 语言等专业分析软件。

## 4. 📱 印刷级高清二维码生成
点击导航栏的 **「二维码生成」**。这是将您的问卷推向受众的关键媒介。

* **自适应部署地址**：二维码解析出的目标链接会自动识别您当前的访问域名和协议（无论是本地 `localhost`、局域网 IP 还是 VPS 外网域名），**实现零配置自适应生成**。
* **双 Canvas 高清印刷级下载**：
  * **前台大方预览**：界面上呈现 290x290 分辨率的轻量化二维码，加载极速，体验流畅。
  * **1024x1024 PNG 导出**：当您点击 **「下载二维码」** 按钮时，系统会自动调用隐式离屏 Canvas，在后台实时生成 **1024x1024 像素的印刷级超清无损 PNG 图片**。您可以直接将导出的超清图片应用于商业海报印刷、户外易拉宝以及各类印刷媒介，边缘毫无毛刺。

## 5. ⚙️ 系统设置与品牌定制
点击导航栏 of **「系统设置」**。这是定义您问卷平台个性的地方。

### 系统品牌与主题色动态更换
* **系统名称**：定制显示在后台左上角及前台页面的系统大标题。
* **系统主题色 (Theme Color Picker)**：
  * 内置颜色选择器。您可以挑选任何符合您企业 VI 的品牌色。
  * 保存设置后，后台管理系统的侧边栏选中态、全局动作按钮、状态标签等都会**动态、无缝更新为您选择的主题色**，呈现极致的高级设计感。
* **系统 Logo**：上传系统品牌 Logo，系统会自动更换后台左上角的标志。
  * **冗余物理自动清理**：当您更换或标记删除系统 Logo 时，系统会在服务器底层自动物理销毁旧的文件，拒绝产生无用占容垃圾。
* **浏览器标题模板**：支持使用通配符定制标签页标题。例如 `{page} - {app}` 将在问卷页自动替换为 `问卷标题 - 我的系统名称`。
* **全局页脚版权信息**：输入您的版权文字（如 `Copyright © 2026 我的公司`），前台展示页和后台管理页脚会同步更新呈现。

### 自适应访问配置
* **站点基础地址**：填写您本地部署或 VPS 部署的公网/局域网根地址（例如 `https://test.gdmrcare.com`）。系统生成的问卷二维码与分享预览连接将以此地址为基准。

## 6. 🛠️ 系统维护与安全建议
1. **数据库一键备份**：
   * 本系统在 `scripts/` 目录下为您提供了一个无物理依赖（无需 `mysqldump`）的 Pure PHP 数据库备份脚本 `backup.php`。
   * 管理员可通过终端运行以下命令，在 `backups/` 目录下一键生成包含完整表结构及数据的 `.sql` 备份快照：
     ```bash
     php scripts/backup.php
     ```
2. **环境健康诊断**：
   * 访问 `/check_env.php` 页面，可一键复核服务器环境的各项健康指标，包括 PHP 版本、PDO 驱动、临时目录写入权限、上传大小限制、安全选项等，所有异常一目了然。
3. **安全上传规范**：
   * 系统在服务器层面已彻底杜绝了 `.svg` 格式文件的上传，预防了任何通过 SVG XML 嵌入脚本的 XSS 跨站攻击风险。
   * **目录权限限制**：请确保 VPS 上的 `public/uploads/` 目录对于 Web 服务进程（如 `nginx` 或 `apache`）具备写入权限（可执行 `chmod -R 777 public/uploads` 进行加固）。
